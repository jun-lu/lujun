DOM 文件夹
	用于构建DOM标准中所定义的各种DOM对象接口，并包含部分说明。
	
	DOM 基本层次结构
		<html>
			<head>
				<title>Hello DOM</title>
			</head>
			<body>
				
			</body>

	DOM 基本对象接口
		
	Node 			//基本节点类型
		Document		//表示文档
			-HTMLDocuemnt	// document是它的实例
		Element		//元素节点
			-HTMLElement	//所有HTML元素都是它或它的子类的实例对象
		CharacterData 	//提供了对字符串操作的方法
			-Text			//文本节点
